# 迷路
簡単なゲームエンジンとそのゲームエンジンを使った簡単なゲームです。
## 操作方法
このゲームはコントローラーとキーボードに対応しています。

アクション | キーボートとマウス | コントローラー
--- | --- | ---
移動 | WASD | 左スティック
カメラ操作 | マウス | 右スティック
カメラ操作 | マウス | 右スティック
終了 | ESC |
